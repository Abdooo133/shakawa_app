## 📥 تنزيل التطبيق

روح على [**آخر إصدار**](../../releases/latest) واختار الملف المناسب لجهازك.

---

### 🤖 Android

1. نزّل ملف **`app-release.apk`**
2. افتحه على تليفونك
3. لو ظهرت رسالة — اضغط **السماح** بتنصيب من مصادر خارجية
4. اضغط **تثبيت** ✅

---

### 🪟 Windows

1. نزّل ملف **`windows-release.zip`**
2. فك الضغط
3. افتح الفولدر وشغّل ملف **`.exe`** ✅

---

### 🍎 macOS

1. نزّل ملف **`macos-release.zip`**
2. فك الضغط
3. شغّل ملف **`.app`**
4. لو ظهرت رسالة أمان — روح **System Settings → Privacy & Security** واضغط **Open Anyway** ✅
